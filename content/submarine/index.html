What is Apache Hadoop Submarine?

Submarine is a new subproject of Apache Hadoop.

Submarine is a project which allows infra engineer / data scientist to run deep learning applications (Tensorflow, Pytorch, etc.) on resource management platform (like YARN).

Scalable

The submarine program supports YARN and Kubernetes as resource scheduling frameworks.

Containers and dockers are supported as containers for machine learning algorithms in YARN.

Compatibility

Supports multiple machine learning frameworks such as tensorflow, Pytorch, MxNet, etc.

Integration

Submarine is not just a machine learning engine.

It covers the entire process of machine learning, from algorithm development, model batch training, model incremental training, model online services, and model management.

Questions and Answers

How can I contribute

The contribution process is the same as for any other Hadoop subproject. The Submarine Contribution Guidelines is available from the wiki.
